# Deployment Guide

This project is deployed on [Vercel](https://vercel.com).

## Steps

1. Push changes to `main`
2. Vercel auto-deploys via GitHub integration
3. Check deployment status at [vercel.com/dashboard](https://vercel.com/dashboard)

## Environment Variables

Set the following in Vercel project settings:

- `NEXT_PUBLIC_API_URL` — base URL of the API
